# Implement

Work from the plan or spec, not memory. Re-read it before starting.

If the plan contradicts what you find in the code, stop and report before proceeding.

Same approach fails 3 times → STOP → REVERT → PIVOT. Three different approaches fail → escalate.

Backward transitions don't invalidate completed work. Update only what's affected.
